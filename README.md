# 魔物獵人 Discord Bot 使用指南

這是魔物獵人 Discord Bot 的文檔倉庫。

## 📚 關於此倉庫

此倉庫只包含文檔內容，用於生成使用指南網站。

## 🌐 網站

文檔網站：https://hurtzqq.github.io/TWCBot-docs/

## 📝 本地建置

如果需要本地預覽：

```bash
# 安裝 MkDocs
pip install mkdocs mkdocs-material mkdocs-minify-plugin

# 啟動本地伺服器
python -m mkdocs serve
```

## 🚀 部署

部署到 GitHub Pages：

```bash
python -m mkdocs gh-deploy
```

## 📖 文檔結構

- `docs/` - 文檔內容
- `mkdocs.yml` - MkDocs 配置檔案
