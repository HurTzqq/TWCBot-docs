    大小金系統

    大小金系統讓您可以管理持有的金冠任務和需求的金冠，方便與其他獵人交換。

    📋 功能概覽

    大小金系統包含以下功能：

    • 📋 金冠持有與需求：管理您持有的金冠和需求的金冠
    • 🌐 公開持有/需求金冠：將您的金冠資訊公開到頻道
    • 🎯 大小金開團：使用持有的金冠建立招募隊伍

    📝 金冠持有與需求

    進入管理介面

    前往「大小金系統」頻道，點擊「📋 金冠持有與需求」按鈕。

    持有金冠管理

    新增持有金冠

    1. 點擊「👑 持有金冠」按鈕
    2. 選擇大小金類型（大金 / 小金）
    3. 選擇魔物
    4. 輸入剩餘次數
    5. 確認後會新增到您的持有列表

    移除持有金冠

    1. 點擊「移除持有金冠」按鈕
    2. 系統會顯示分頁列表（每頁 15 個）
    3. 使用「上一頁」/「下一頁」瀏覽
    4. 從下拉選單選擇要移除的金冠
    5. 確認後會從列表中移除

    需求金冠登記

    新增需求金冠

    1. 點擊「🆘 需求金冠」按鈕
    2. 選擇大小金類型（大金 / 小金）
    3. 選擇魔物
    4. 確認後會新增到您的需求列表

    移除需求金冠

    1. 點擊「移除需求金冠」按鈕
    2. 系統會顯示分頁列表（每頁 15 個）
    3. 使用「上一頁」/「下一頁」瀏覽
    4. 從下拉選單選擇要移除的需求
    5. 確認後會從列表中移除

    返回主選單

    在移除介面中，點擊「按一下返回」按鈕可以返回主選單。

    🌐 公開持有/需求金冠

    功能說明

    將您的持有金冠和需求金冠公開到「大小金互助」頻道，讓其他獵人可以看到。

    使用方式

    1. 前往「大小金互助」頻道
    2. 點擊「公開持有/需求金冠」按鈕
    3. 系統會發送一個訊息，顯示：
    4. 您持有的所有金冠及剩餘次數
    5. 您需求的所有金冠

    顯示格式

    👑 持有金冠
大金◈影蜘蛛 ʟᴇꜰᴛ 1
大金◈泡狐龍 ʟᴇꜰᴛ 2
小金◈護兇爪龍 ʟᴇꜰᴛ 1

🆘 需求金冠
大金◈千刃龍
小金◈刺花蜘蛛

    🔍 大小金自動配對

    功能說明

    系統會自動為需求金冠的玩家配對持有該金冠的玩家。

    配對條件

    1. 有需求金冠：您必須先登記需求金冠
    2. 其他玩家持有：其他玩家持有您需求的金冠
    3. 無償分享設定：如果其他玩家設定了「無償分享」，即使您沒有需求金冠也會被配對

    配對通知

    配對成功後，系統會發送通知訊息，包含： - 需求者資訊 - 需求的金冠 - 可交換的玩家列表 - 如果對方設定無償分享，會特別標註

    無償分享設定

    1. 在「大小金系統」中點擊「🔔 大小金通知設定」按鈕
    2. 開啟「無償分享」開關
    3. 開啟後，即使您沒有需求金冠，系統也會在配對時考慮您

    📊 查看公開資訊

    在大小金互助頻道

    您可以查看其他玩家公開的持有/需求金冠資訊，方便尋找交換對象。

    💡 小技巧

    1. 定期更新：及時更新持有金冠的剩餘次數
    2. 清楚登記需求：登記您真正需要的金冠
    3. 使用公開功能：讓其他獵人更容易找到您
    4. 開啟無償分享：如果願意無償分享，可以開啟此功能

    ⚠️ 注意事項

    資料管理

    • 需求金冠超過 7 天會自動清除
    • 持有金冠的剩餘次數需要手動更新
    • 移除金冠會直接從資料庫刪除

    自動配對

    • 配對是自動進行的，不需要手動操作
    • 如果沒有找到配對對象，不會發送通知
    • 無償分享功能需要手動開啟

    🔄 資料同步

    • 所有資料都儲存在資料庫中
    • 多個頻道可以同時查看您的資料
    • 資料變更會即時反映

    ❓ 常見問題

    Q: 需求金冠會一直存在嗎？

    A: 不會，超過 7 天會自動清除。

    Q: 如何更新持有金冠的剩餘次數？

    A: 需要先移除舊的記錄，然後重新新增正確的次數。

    Q: 無償分享是什麼？

    A: 開啟後，即使您沒有需求金冠，系統也會在配對時考慮您，讓其他玩家知道您願意無償分享。

    Q: 可以同時持有和需求同一個金冠嗎？

    A: 可以，系統允許這樣的情況。
