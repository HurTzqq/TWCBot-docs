# 快速開始

歡迎使用魔物獵人 Discord Bot！本指南將幫助您快速了解如何使用這個 Bot。

## 📋 前置準備

在使用 Bot 之前，請確保您已經：

1. ✅ 加入包含此 Bot 的 Discord 伺服器
2. ✅ 擁有 Discord 帳號並已登入
3. ✅ 了解基本的 Discord 操作

## 🎯 第一步：新血報到

**這是使用所有功能的前提！**

### 使用方式

1. 在新血報到頻道中點擊〖開始新血報到按鈕〗
2. 按照系統提示填寫以下資訊：
   - **擅長武器**：可選擇多種武器
   - **出沒時間**：您的遊戲時間
   - **遊玩平台**：PC / PS / Xbox 等
   - **習慣交流方式**：語音通話 / 文字聊天等
3. 最後會要求輸入：
   - **遊戲名稱**
   - **遊戲 ID**（只能包含英文字母和數字，會自動轉為大寫）

### 注意事項

- ⚠️ 遊戲 ID 只能包含英文字母和數字
- ⚠️ 遊戲 ID 會自動轉換為大寫
- ⚠️ 所有欄位都有字數限制，請注意

完成註冊後，您就可以使用所有功能了！

## 🎮 主要功能頻道

Bot 會在以下頻道提供功能：

### 1. 新血報到頻道
- 完成玩家註冊
- 查看自己的註冊資料

### 2. 獵友招募頻道
- 建立招募隊伍
- 管理自己的隊伍
- 查看招募公告

### 3. 大小金系統頻道
- 管理持有/需求金冠
- 查看公開的金冠資訊
- 開啟大小金開團

### 4. 大小金互助頻道
- 查看大小金開團通知
- 查看公開的持有/需求金冠

## 🔍 查詢自己的資料

使用 `/查詢資料` 指令可以隨時查看自己的註冊資訊，包括：
- 遊戲名稱和 ID
- 出沒時段
- 遊玩平台
- 交流方式
- 擅長武器
- 註冊日期
