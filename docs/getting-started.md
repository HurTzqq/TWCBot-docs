## 🎯 第一步：新血報到 / Registration

**完成新血報到是使用所有功能的前提！**

### 🔹 使用方式

1. 前往新血報到頻道，點擊 **〖開始新血報到〗** 按鈕  
2. 按照提示填寫資訊：
   - **擅長武器**：可選多種武器  
   - **出沒時間**：遊玩時間  
   - **遊玩平台**：PC / PS / Xbox 等  
   - **交流方式**：語音 / 文字聊天  
3. 最後輸入：
   - **遊戲名稱**  
   - **遊戲 ID**（僅限英文字母與數字，會自動轉大寫）

### ⚠️ 注意事項

- 遊戲 ID 僅可使用英文字母與數字  
- 遊戲 ID 會自動轉為大寫  
- 所有欄位均有字數限制  

完成後即可使用 Bot 的全部功能。

## 🎮 主要功能頻道 / Channels

### 1. 📝 新血報到頻道
- 完成玩家註冊  
- 查看自己的註冊資料  

### 2. 👥 獵友招募頻道
- 建立招募隊伍  
- 管理自己的隊伍  
- 查看招募公告  

### 3. 👑 大小金系統頻道
- 管理持有/需求金冠  
- 查看公開金冠資訊  
- 開啟大小金開團  

### 4. 🔄 大小金互助頻道
- 查看開團通知  
- 查看公開的持有/需求金冠
