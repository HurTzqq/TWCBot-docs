## 🐞 回報問題 / Bug Report

### 🔧 回報前請準備：

1. **問題描述**：發生了什麼  
2. **操作步驟**：做了哪些操作  
3. **預期行為**：正常程序應該是什麼狀況  
4. **實際行為**：實際發生了什麼狀況  
5. **截圖**：能提供截圖就更好處理了

### 📌 回報範例


```
問題：無法加入隊伍
操作：點擊「加入隊伍」按鈕
預期：正常來說要加入隊伍
實際：顯示「您已經在其他隊伍中」
截圖：![](./assets/截圖.png)
```
