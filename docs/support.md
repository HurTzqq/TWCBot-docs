# 技術支援

如果您在使用 Bot 時遇到問題，這裡提供了一些獲取幫助的方式。

## 🆘 獲取幫助

### 1. 查看文檔

首先，請查看相關的使用指南：
- [快速開始](getting-started.md)
- [功能指南](features/registration.md)
- [常見問題](faq.md)

### 2. 檢查常見問題

許多問題都可以在 [常見問題](faq.md) 中找到答案。

### 3. 聯繫管理員

如果問題仍然無法解決，請聯繫伺服器管理員：
- 在 Discord 伺服器中發送訊息
- 提供詳細的問題描述
- 附上相關的錯誤訊息或截圖

---

## 🐛 回報問題

### 回報前請準備

1. **問題描述**：清楚描述您遇到的問題
2. **操作步驟**：說明您做了什麼操作
3. **預期結果**：您期望發生什麼
4. **實際結果**：實際發生了什麼
5. **錯誤訊息**：如果有錯誤訊息，請附上
6. **截圖**：如果可能，請附上截圖

### 回報格式範例

```
問題：無法加入隊伍
操作：點擊「加入隊伍」按鈕
預期：成功加入隊伍
實際：顯示「您已經在其他隊伍中」
錯誤訊息：無
截圖：[附上截圖]
```
